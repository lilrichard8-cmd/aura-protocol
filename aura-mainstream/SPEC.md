# AURA 主流版前端 UI 规格

## 设计参考
- **Threads** × **小红书（RED）** 混合风格
- 混合内容瀑布流，支持图文/视频/纯文字/直播卡片混排

## 技术栈
- Vite + React + TypeScript + Tailwind CSS
- Mobile-first（375px基准），响应式适配桌面

## 色彩体系
- 主色：#7C3AED（紫色，AURA品牌色）
- 背景：#FAFAFA（浅灰白）
- 卡片：#FFFFFF
- 文字：#1A1A1A / #6B7280（次要）
- 强调：#F59E0B（ORA金色，用于代币相关）
- 暗色模式支持

## 页面结构

### 1. 首页 Feed
- **双栏瀑布流**（小红书风格）
- 卡片类型：
  - 📷 图文卡片：封面图 + 标题 + 作者头像/名字 + ❤️数
  - 🎬 视频卡片：封面帧 + 播放按钮 + 时长 + 作者 + ❤️数
  - 💬 文字卡片：正文预览(3行) + 作者 + ❤️数（Threads风格，紧凑矮卡片）
  - 🔴 直播卡片：直播封面 + LIVE标签 + 观看人数
- 卡片高度不固定，根据内容自适应（masonry布局）
- 下拉刷新 + 无限滚动

### 2. 发现页 Explore
- 顶部搜索栏
- 热门标签横滑
- 推荐创作者轮播
- 分类Tab：全部 / 摄影 / 视频 / 文字 / 音乐 / 插画

### 3. 发布页 Create
- 底部Tab栏中央的 + 按钮
- 支持：图文笔记 / 短视频 / 纯文字 / 开启直播
- 图文编辑器：多图上传 + 滤镜 + 标签 + 正文
- 发布时可选择是否启用策展挖矿

### 4. 消息页 Messages
- 对话列表
- 一对一聊天
- 支持发送ORA打赏

### 5. Profile页
- 头像 + 用户名 + Bio
- Creator Coin信息栏（持有者数量/价格/铸造按钮）
- 关注/粉丝数
- 内容Tab：帖子（网格）/ 帖子（列表）/ 策展记录 / 喜欢
- 编辑Profile按钮

### 6. 详情页
- **图文**：左右滑图片 + 下方正文 + 评论区（小红书风格）
- **视频**：全屏播放 + 右侧互动栏（❤️/评论/分享/策展）
- **文字**：对话串 + 引用回复（Threads风格）

## 底部导航栏（5个Tab）
```
🏠 首页 | 🔍 发现 | ➕ 发布 | 💬 消息 | 👤 我的
```

## Web3 元素（自然融入，不突兀）
- 右上角ORA余额小组件（点击展开钱包）
- Creator Coin在Profile页展示
- 策展挖矿入口：长按内容 → "策展" 选项
- 打赏按钮在评论区旁边
- 首次使用时自动创建嵌入式钱包（无需用户理解区块链）

## 关键交互
- 双击卡片 = 点赞
- 长按 = 更多选项（策展/举报/分享）
- 右滑返回
- 下拉刷新带AURA Logo动画
