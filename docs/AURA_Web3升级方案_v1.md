## 升级概览

### 当前状态
AURA 已具备：
- ✅ Solana 智能合约（用户、内容、金库、市场、治理）
- ✅ Arweave 永久存储
- ✅ 95% 创作者分成
- ✅ 7 天锁仓期（争议追回）
- ✅ DAO 治理框架（5 委员会）
- ✅ 二创许可分账系统

### 升级目标
从「平台」进化为「协议」：
- 🎯 用户拥有自己的社交图谱
- 🎯 内容可被其他应用组合调用
- 🎯 早期发现者获得激励
- 🎯 创作者可发行个人代币
- 🎯 无需许可的生态系统

### 3️⃣ 策展挖矿 (Curation Mining)

**核心理念:** 早期发现好内容的人应该获得奖励

**机制设计:**
```
时间轴:
────────────────────────────────────────────►
     │                    │                 │
   内容发布            热门阈值           结算
     │                    │                 │
  策展人 A 点赞      策展人 B 点赞      奖励分配
  (权重高)           (权重低)
```

**开发任务:**

| 任务 | 描述 | 工时 | 优先级 |
|------|------|------|--------|
| 3.1 | 设计策展积分算法 | 3天 | P0 |
| 3.2 | 开发策展记录合约 | 5天 | P0 |
| 3.3 | 实现时间衰减权重 | 2天 | P0 |
| 3.4 | 策展奖励池和分配 | 4天 | P0 |
| 3.5 | 策展排行榜 UI | 3天 | P1 |
| 3.6 | 防刷机制 | 4天 | P1 |

**总工时:** 21 天

**算法设计:**
```
策展得分 = f(发现时间, 内容最终热度)

发现时间权重:
- 前 1 小时发现: 10x
- 1-6 小时: 5x  
- 6-24 小时: 2x
- 24-72 小时: 1x
- 72 小时后: 0.1x

策展奖励池:
- 来源: 内容产生收益的 5%
- 分配: 按策展得分比例分配给所有早期策展人
```

**智能合约:**

## 改进版 Tokenomics

### 当前问题

1. **$ORA 主要用于治理**，缺乏日常使用场景
2. **销毁来源有限**，通缩压力不足
3. **早期用户激励不够**，冷启动困难
4. **没有与创作者代币的联动**

### 改进方案

#### 1. 代币用途扩展

```
┌─────────────────────────────────────────────────────────┐
│                    $ORA 使用场景                         │
├─────────────────────────────────────────────────────────┤
│ 【治理】                                                │
│  - 质押 100+ $ORA 获得投票权                            │
│  - 委员会选举投票                                        │
│  - 协议参数修改投票                                      │
├─────────────────────────────────────────────────────────┤
│ 【策展挖矿】← 新增                                       │
│  - 质押 $ORA 增加策展权重                                │
│  - 策展奖励以 $ORA 发放                                  │
├─────────────────────────────────────────────────────────┤
│ 【创作者代币】← 新增                                     │
│  - 创建 Creator Coin 需燃烧 1000 $ORA                   │
│  - Creator Coin 交易手续费的 50% 用于回购销毁 $ORA       │
├─────────────────────────────────────────────────────────┤
│ 【内容碎片化】← 新增                                     │
│  - 碎片化操作费用以 $ORA 支付                            │
│  - 碎片治理投票需持有 $ORA                               │
├─────────────────────────────────────────────────────────┤
│ 【协议费用】                                             │
│  - NFT 铸造费（100% 销毁）                               │
│  - 平台手续费（部分销毁）                                │
│  - 广告费（50% 销毁）                                    │
├─────────────────────────────────────────────────────────┤
│ 【质押收益】← 新增                                       │
│  - 质押 $ORA 获得协议收入分红                            │
│  - 质押者共享 50% 的协议手续费收入                       │
└─────────────────────────────────────────────────────────┘
```

#### 2. 新版代币分配

```
总供应量: 20 亿 $ORA (硬顶，不再是软顶)

┌────────────────────────────────────────────────────────┐
│                    代币分配                             │
├──────────────────┬─────────┬───────────────────────────┤
│ 类别             │ 比例    │ 数量 / 备注               │
├──────────────────┼─────────┼───────────────────────────┤
│ 创作者激励       │ 40%     │ 8 亿，10 年线性释放       │
│ 策展者激励       │ 10%     │ 2 亿，按策展贡献分配      │
│ 社区金库 (DAO)   │ 20%     │ 4 亿，DAO 投票决定使用    │
│ 团队             │ 15%     │ 3 亿，4 年归属 + 1 年锁定 │
```

#### 4. 质押机制

```
┌─────────────────────────────────────────────────────────┐
│                    $ORA 质押系统                         │
├─────────────────────────────────────────────────────────┤
│ 【基础质押】                                             │
│  - 最低质押: 100 $ORA                                   │
│  - 解锁期: 7 天                                          │
│  - 收益: 协议手续费收入的 50%，按质押比例分配            │

│ 【策展质押】                                             │
│  - 质押越多，策展权重越高                                │
│  - 权重公式: base_weight * (1 + log(staked_amount/100)) │
│  - 鼓励长期质押，降低投机                                │
├─────────────────────────────────────────────────────────┤
│ 【治理质押】                                             │
│  - 投票权 = 质押数量 * 质押时长系数                      │
│  - 1 个月: 1x                                           │
│  - 3 个月: 1.2x                                         │
│  - 6 个月: 1.5x                                         │
│  - 12 个月: 2x                                          │
└─────────────────────────────────────────────────────────┘
```
