# AURA After Dark 前端 UI 规格

## 设计参考
- **Fansly** 为主要参考
- 订阅制创作者平台，支持分层订阅/PPV/直播/私信

## 技术栈
- 同主流版：Vite + React + TypeScript + Tailwind CSS
- 仅PWA分发（不上App Store）
- Mobile-first

## 色彩体系
- 主色：#1A1A2E（深蓝黑）
- 次色：#E94560（玫红强调色）
- 背景：#16213E（深色）
- 卡片：#1A1A2E / #0F3460
- 文字：#EAEAEA / #A0A0A0（次要）
- ORA金色：#F59E0B（代币相关）
- 全暗色设计，无亮色模式

## 页面结构

### 1. 首页 Feed
- 双Feed切换：
  - **关注** — 已订阅创作者的更新流
  - **发现 (For You)** — 算法推荐（Fansly风格）
- 卡片类型：
  - 免费预览卡（正常显示）
  - 付费内容卡（模糊遮罩 + 🔒解锁按钮 + 价格）
  - 直播卡（LIVE标签 + 观看人数）
- 瀑布流布局

### 2. 创作者主页
- 封面大图 + 头像 + Bio
- **订阅层级展示**（Fansly核心特色）：
  ```
  🆓 免费关注 — 基础内容
  🥉 铜牌 $5/月 — 独家图文
  🥈 银牌 $15/月 — 完整内容 + 私信
  🥇 金牌 $30/月 — 一切 + 定制内容
  ```
- 每层显示解锁的内容数量
- Creator Coin信息 + 购买按钮
- 内容网格（锁定内容显示模糊预览）

### 3. 内容详情
- 免费内容：正常展示
- 付费内容：
  - 未解锁 → 模糊预览 + 价格 + 解锁按钮
  - 已解锁 → 正常展示
- PPV（单条付费）支持
- 水印自动叠加（创作者可配置）

### 4. 私信 DM
- 免费消息 / 付费消息（创作者设定价格）
- PPV消息（发送付费内容）
- 打赏按钮
- 语音消息支持

### 5. 直播
- 全屏直播画面
- 弹幕 + 打赏动画
- 票务直播（付费入场）
- 私人直播（指定订阅层级可看）
- 打赏目标进度条

### 6. 发布页
- 发布内容时选择：
  - 可见范围（免费 / 铜牌+ / 银牌+ / 金牌）
  - 是否启用PPV（单独定价）
  - 是否添加水印
  - 定时发布
- 支持图片/视频/图集

### 7. 钱包/收益
- ORA余额 + 收益统计
- 订阅者管理
- 提现（链上转账）
- 收益图表（日/周/月）

## 底部导航栏
```
🏠 首页 | 🔍 发现 | ➕ 发布 | 💬 私信 | 👤 我的
```

## 年龄验证
- 首次访问弹窗确认18+
- KYC验证创作者身份（发布内容必须）

## 隐私/安全
- 地理封锁（创作者可屏蔽特定国家/地区）
- 截图检测提醒（PWA能力有限，但可以禁用右键）
- 自动水印（用户名+日期）
- 内容存储在Arweave上，永久不可删除（创作者需理解这一点）

## Web3 元素
- 加密支付为主（ORA/USDC/SOL）
- 嵌入式钱包自动创建
- 订阅 = 链上交易（但UI包装成"订阅"按钮，用户无感）
- Creator Coin持有者专属内容层级
